# Apex Netstats!

Electron app to display network statistics for **Apex Legends**


# Clone and Build

**Clone Repo:** git clone https://github.com/akshayha/apex-netstats.git
**Yarn Install:** yarn
**Quasar Build:** quasar build -m electron --target win32
